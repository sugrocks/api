Introduction

Welcome to the /sug/.rocks API!

You can use our API to get the recent /sug/ threads, OP template, latest leaks (if available) and downloads !

You can also check out the Apiary version of this API.

  • Not included: TV ratings (like [TV-PG])
  • Added: timestamp

HTTP Request

GET https://api.sug.rocks/schedule.json

This endpoint is using the American Eastern timezone (EST, -0500). Use the epoch `timestamp` to convert to your own (local) timezone if needed.

HTTP Request

GET https://api.sug.rocks/dl.json

Files that contains two episodes in one have an underscore between episodes number. Like for episode 1 & 2: `1_2`.
`url`, `marebucks`, `dailymotion` and `torrent` can be “null” if not available.
